# Movie Explorer App

## About
This project is a simple web application that uses a public API to fetch and display movie data. Users can search for movies and view basic details like title, year, and poster.

## API Used
OMDb API (https://www.omdbapi.com/)

## Features (Planned)
- Search movies by name
- Filter results
- Sort results
- Responsive design

## Technologies
- HTML
- CSS
- JavaScript
